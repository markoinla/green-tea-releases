# Competitor Analysis

Analyze and compare competitor products, features, pricing, and market positioning.

## When to use

Use this skill when the user wants to:
- Research competitor products or services
- Compare features across competing solutions
- Analyze market positioning and differentiation
- Create competitive landscape summaries
- Identify strengths, weaknesses, and gaps in competitor offerings

## Instructions

When performing competitor analysis:

1. **Gather information** — Use web search to find current information about the competitors mentioned. Look for product pages, pricing, feature lists, and recent news.

2. **Structure the analysis** — Organize findings into clear categories:
   - Product overview and positioning
   - Key features and capabilities
   - Pricing and plans
   - Target audience
   - Strengths and weaknesses

3. **Create comparison tables** — When comparing multiple competitors, use markdown tables to make differences clear at a glance.

4. **Identify insights** — Go beyond listing features. Highlight:
   - Gaps in the market that no competitor addresses
   - Common patterns across competitors
   - Unique differentiators for each product
   - Potential opportunities

5. **Write to notes** — Save the analysis as a structured document in the user's notes using the notes tools. Use clear headings and bullet points for scannability.

## Output format

Structure the output as a document with:
- Executive summary (2-3 sentences)
- Individual competitor profiles
- Comparison table
- Key takeaways and recommendations
